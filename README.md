# englishtowns
Generate fake English town names

### Overview
A function for creating fake English town names.

### Installation
``` 
# install.packages("devtools")
devtools::install_github("sbha/englishtowns")
```

### Usage
```
library(englishtowns)
generate_town()
#> Greathamrough
```
